Machine Learning Compatible Computer Graphics

This project explores the synergy between high‑performance rendering and machine learning.

  • I built a modular game engine with an OpenGL backend and a clean abstraction layer.
  • Non‑player characters learn behaviours via reinforcement learning, enabling adaptive AI.
  • The engine emphasises clean architecture, extensibility and the separation of concerns.

The project demonstrates how computer graphics and machine learning can coexist in real time.
