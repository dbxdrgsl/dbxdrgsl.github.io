Peripheral Devices

An introductory resource on input, output and storage peripherals, based on the Paradis Cambridge School curriculum.

Introduction

Peripheral devices are external or semi‑external components that connect to a computer’s central unit. They allow users to input data (e.g. keyboards, mice), output results (e.g. monitors, printers), store information (e.g. USB drives) or even perform both input and output simultaneously (e.g. touchscreens).

Understanding how these devices communicate with the processor and memory lays the foundation for effectively using and troubleshooting computer systems.

Lesson Outline

The following summary is adapted from the Proiect didactic – Dispozitive periferice document prepared for class IX at Paradis Cambridge School. It identifies the main learning objectives, teaching methods and content to be covered.

Objectives

  • Define the concept of a peripheral device.
  • Classify peripherals by function: input, output, storage and input‑output.
  • Provide concrete examples for each category and describe their roles.
  • Explain how peripherals interact with the central processing unit (CPU) and memory.

Key Content

Peripherals can be grouped into four broad categories:

  • Input devices: keyboard, mouse, scanner, microphone, joystick.
  • Output devices: monitor, printer, speakers, video projector.
  • Storage devices: USB sticks, external hard drives, optical media (CD/DVD), memory cards.
  • Input‑output devices: touchscreen displays, multifunction printers, modems.

When you press a key on the keyboard, an electrical signal representing the character’s code is sent to the CPU via the input subsystem. The processor interprets this code, stores it temporarily in RAM and instructs an output device (usually the monitor) to display the corresponding symbol. Similar exchanges occur between all peripherals and the CPU, coordinated by device drivers and communication protocols such as USB and Bluetooth.

Evaluation Sheet (Sample)

The following questions are adapted from the evaluation sheet associated with this lesson. They are intended for individual assessment and can be answered in class or as homework.

  1. Choose the correct option: input peripherals are used for …
    a) displaying results; b) introducing data into the computer; c) storing data; d) printing documents.
  2. Match each device with its category (input / output / storage):
    a) printer – _____
    b) mouse – _____
    c) USB stick – _____
    d) monitor – _____
  3. Fill in the blanks: a scanner is an ________ device, whereas a video projector is an ________ device.
  4. Select the correct answer: which of the following is an input‑output device? (a) Keyboard; (b) Touchscreen; (c) Simple printer; (d) Speakers.
  5. A student claims that “all peripheral devices are necessarily external”. Argue whether the statement is correct or incorrect and give an example.
  6. Explain how a peripheral can belong to two categories at the same time and provide a concrete example.

Proposed Timing (50 minutes)

To keep the lesson within a 50 minute window for a class of approximately 20 pupils, the activities may be allocated as follows:

Activity Duration
Organisational moment & recap of previous concepts 5 min
Brainstorming: personal peripheral devices and usage 5 min
Presentation of definitions and classification of peripherals 15 min
Discussion of information flow & diagram analysis 10 min
Individual or pair work on evaluation sheet 10 min
Conclusion, feedback & homework instructions 5 min
